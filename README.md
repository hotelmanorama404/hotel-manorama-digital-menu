This is a QR-based digital menu website for Hotel Manorama, a family restaurant.

Customers can scan a QR code and view the full menu on their mobile phone without installing any app.

## Features
- Mobile-friendly design
- QR code compatible
- Category-wise food menu
- Veg and Non-Veg indicators
- Click-to-call phone number
- Smooth scrolling navigation

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open index.html in a browser  
2. Or host it online (GitHub Pages / Netlify)
3. Generate a QR code for the website link
4. Place the QR code on tables
